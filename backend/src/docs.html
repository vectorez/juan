API Reference

API REST pública para consultar datos de facturación y recaudos municipales.

BASE URL https://tu-dominio.replit.app/api/public
Primeros pasos
Sin autenticación requerida. Todos los endpoints son públicos y de solo lectura. La API devuelve JSON en todas las respuestas.

La API permite consultar los datos de municipios almacenados en el sistema, incluyendo datos de facturación y recaudos, importaciones y estadísticas agregadas.

Ejemplo rápido — cURL 
# Listar municipios disponibles
curl "https://tu-dominio.replit.app/api/public/municipios"
Formato de respuesta

Todas las respuestas siguen el mismo esquema base:

JSON
{
  "success": true,          // boolean — éxito o error
  "total": 42,              // integer — total de registros (cuando aplica)
  "pagination": {          // objeto de paginación (cuando aplica)
    "page": 1,
    "limit": 100,
    "pages": 5,
    "has_next": true,
    "has_prev": false
  },
  "data": [...]            // array o objeto con los datos
}
Manejo de errores
CódigoSignificadoDescripción
200OKRespuesta exitosa
404Not FoundMunicipio o recurso no existe
500Server ErrorError interno del servidor

Ejemplos — JavaScript / Fetch
JavaScript
const BASE = 'https://tu-dominio.replit.app/api/public';

// Listar municipios
const municipios = await fetch(`${BASE}/municipios`)
  .then(r => r.json());

// Obtener facturación con paginación
const params = new URLSearchParams({ page: '1', limit: '50' });
const facturacion = await fetch(`${BASE}/municipios/apartado/facturacion?${params}`)
  .then(r => r.json());

console.log(`Total registros: ${facturacion.pagination.total}`);
facturacion.data.forEach(row => console.log(row));
Ejemplos — Python
Python
import requests

BASE = 'https://tu-dominio.replit.app/api/public'

# Listar municipios
r = requests.get(f'{BASE}/municipios')
municipios = r.json()['data']

# Obtener estadísticas
stats = requests.get(
    f'{BASE}/municipios/apartado/stats',
    params={'tipo': 'facturacion', 'columna': 'ciclo'}
).json()

# Paginar todos los registros
def get_all_pages(slug, tabla):
    page, all_rows = 1, []
    while True:
        data = requests.get(
            f'{BASE}/municipios/{slug}/{tabla}',
            params={'page': page, 'limit': 1000}
        ).json()
        all_rows.extend(data['data'])
        if not data['pagination']['has_next']:
            break
        page += 1
    return all_rows
Ejemplos — cURL
Shell
# Listar todos los municipios
curl "https://tu-dominio.replit.app/api/public/municipios"

# Filtrar por departamento
curl "https://tu-dominio.replit.app/api/public/municipios?departamento=Antioquia"

# Obtener detalle de un municipio
curl "https://tu-dominio.replit.app/api/public/municipios/apartado"

# Facturación página 2 con 50 registros
curl "https://tu-dominio.replit.app/api/public/municipios/apartado/facturacion?page=2&limit=50"

# Buscar por columna específica
curl "https://tu-dominio.replit.app/api/public/municipios/apartado/facturacion?columna=ciclo&q=01"

# Estadísticas con análisis de columna
curl "https://tu-dominio.replit.app/api/public/municipios/apartado/stats?tipo=facturacion&columna=ciclo"
